# PlatinumTrade.Docs

Official documentation for the **Platinum Trade** ecosystem, including
the **Platinum Trade** desktop application and the **Pt.Okx.Sdk**
development toolkit.

The documentation is built with **Docusaurus** and is automatically
deployed to **GitHub Pages**.

## 🚀 Local Development

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm start
```

Build the production website:

```bash
npm run build
```

------------------------------------------------------------------------

## 📦 Deployment

Deployment is fully automated using GitHub Actions.

Every push to the `main` branch will trigger a new documentation build
and publish the latest version to GitHub Pages.

No manual deployment is required.
