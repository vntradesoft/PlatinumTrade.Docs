# Keyboard Shortcuts

Platinum Trade provides a wide range of keyboard shortcuts and mouse gestures to help you navigate the platform, analyze charts, and execute trades more efficiently.

## General Navigation

| Shortcut | Action |
| :--- | :--- |
| <kbd>F11</kbd> | Toggle Fullscreen mode |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>O</kbd> | Open Options & Settings |
| <kbd>F1</kbd> | Show About/Help information |
| <kbd>Alt</kbd>+<kbd>F4</kbd> | Exit Application |

## Chart View & Zoom

| Shortcut | Action |
| :--- | :--- |
| <kbd>Ctrl</kbd>+<kbd>+</kbd> | Zoom In |
| <kbd>Ctrl</kbd>+<kbd>-</kbd> | Zoom Out |
| <kbd>Ctrl</kbd>+<kbd>0</kbd> | Reset Zoom to default |
| <kbd>Alt</kbd>+<kbd>1</kbd> | Switch to Bar Chart |
| <kbd>Alt</kbd>+<kbd>2</kbd> | Switch to Candlestick Chart |
| <kbd>Alt</kbd>+<kbd>3</kbd> | Switch to Line Chart |

## Trading & Data Management

| Shortcut | Action |
| :--- | :--- |
| <kbd>Ctrl</kbd>+<kbd>N</kbd> | Add a new Symbol to Market Watch |
| <kbd>Ctrl</kbd>+<kbd>B</kbd> | Open Bid/Ask (Order Book) Chart |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>H</kbd> | Open Data History Manager |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>I</kbd> | Open Indicator Plugin Manager |

## Chart Drawing Tools & Interactions

| Shortcut / Gesture | Action |
| :--- | :--- |
| <kbd>Ctrl</kbd>+<kbd>C</kbd> | Copy selected drawing object |
| <kbd>Ctrl</kbd>+<kbd>V</kbd> | Paste copied drawing object |
| <kbd>Del</kbd> | Delete selected drawing object(s) |
| **Double Click** | Edit drawing properties |
| **Ctrl + Click** | Select multiple drawings |
| **Click in empty space** | Unselect all drawings |

---

*Tip: Familiarizing yourself with these shortcuts can significantly speed up your technical analysis and operational workflows during live trading.*
