# Getting Started

Use this guide to quickly launch the desktop application and verify your connection status.

## Prerequisites

- Platinum Trade GUI App installed.
- Valid OKX API credentials (it is highly recommended to start with Demo).

## Launching the Application

Open the Platinum Trade App using the desktop shortcut icon or from the Start Menu.

Upon launch, you will see the main menu, toolbar, the Market Watch panel on the left, and workspace tabs for charting/trading.

## Initial Setup Workflow

1. Open options via Tools -> Options.
2. Select the target environment (Demo or Production) under [API settings](./settings/api-credentials.md).
3. Input your API Key, Secret Key, and Passphrase.
4. Click Test Connection.
5. Save the configuration.

## Recommended Initial Actions

1. Start in Demo mode.
2. Add a symbol (e.g., BTC-USDT) to the Market Watch panel.
3. Open a chart and switch timeframes.
4. Apply a built-in indicator.
5. Test the drawing tools and alert triggers.
