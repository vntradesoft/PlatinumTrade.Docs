# Drawing Tools

Drawing tools help annotate charts and visualize trade setups.

## Available Toolbar Tools

- Horizontal line
- Vertical line
- Trend line
- Rectangle
- Fibonacci retracement
- Measurement
- Text
- Emoji marker

## Basic Workflow

1. Select a drawing tool from the toolbar.
2. Click or drag on the chart to place/create the drawing object.
3. Customize position or style using the object's interactive control points.

## Customizing Indicator Display

To customize display attributes at the indicator level (line style, fill regions, bounds), use the indicator properties dialog.
