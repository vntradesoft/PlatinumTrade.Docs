# Overview

The Platinum Trade GUI App provides a premium, desktop-native trading experience on OKX, featuring high-performance charting, custom technical indicators, strategy backtesting/simulations, and license-based feature tiering.

## Core Capabilities

- Secure API credential configuration with built-in connection diagnostics.
- Real-time strategy execution in both paper trading and live production modes (subject to license tiers).
- Localization support with complete English and Vietnamese language resources.
- Automated client updates powered by Velopack bootstrap hooks.
