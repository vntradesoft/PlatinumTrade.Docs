# Network

This section describes how to configure network settings, including proxies and connection timeouts.

![Network Settings](/img/products/gui/settings/network-settings.png)

## Proxy Server

If your local network or country restricts access to OKX endpoints, you can route all API traffic through a custom proxy server.

- **Enable Proxy**: Check this box to route API requests through your configured proxy.
- **Address**: The IP address or hostname of your proxy server.
- **Port**: The port number of your proxy server.
