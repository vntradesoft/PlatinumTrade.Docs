# License

This section outlines how to input and verify your Platinum Trade software license.

![License Settings](/img/products/gui/settings/license-settings.png)

## License Configuration

To activate your Pro license, complete the following steps in the **License** settings tab:

1. **Hardware ID Registration**: Locate your unique **Machine ID** text box. Click the **Copy** button to copy it to your clipboard, or **Export File** to save it to your Documents directory.
2. **Request License File**: Send this Machine ID to the support team. They will generate and return a signed license file (`.lic`).
3. **Import License**: Click the **Import License File** button and select the `.lic` file you received.
4. **Validation**: The system will automatically validate the license signature and display your license tier, expiration date, and active features. Click **Refresh** to force a validation check at any time.
