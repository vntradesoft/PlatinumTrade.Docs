# License

This section outlines how to input and verify your Platinum Trade software license.

![License Settings](/img/products/gui/settings/license-settings.png)

## License Configuration

- **License Key**: Enter the license key provided to you upon purchase.
- The system will validate the key against your Hardware ID and display the validity status, expiration date, and enabled features.
