# Appearance

This section describes how to configure the appearance of the application, including themes, accent colors, and layout options.

![Appearance Settings](/img/products/gui/settings/apearance-settings.png)

## General Display

- **Theme**: Switch between `Light`, `Dark`, or `Auto`.
- **Accent Color**: Personalize the primary color used for highlights and buttons (`Blue`, `Green`, `Lime`, `Red`, `Orange`, `Purple`).
- **Language**: Choose your preferred interface language (e.g., `English` or `Vietnamese`). Changes apply immediately.
- **Display Time Zone**: Select the time zone you want to use for displaying chart times and trade logs.
- **Follow System Theme**: Enable this to automatically match the app theme to your Windows system theme.

## Sound Settings

- **Alert Sound**: Enable or disable the standard system beep sound when notifications and alerts are triggered.
