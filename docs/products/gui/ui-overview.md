# UI Overview

The main desktop window of the Platinum Trade App is organized into four main functional zones.

## 1. Top Menu Bar

- Includes: File, Edit, View, Insert, Charts, Tools, Help.
- Supports standard keyboard shortcuts for common actions (e.g., Ctrl+N, F11, Ctrl+Shift+O).

## 2. Main Toolbar

- Access buttons for adding symbols, selecting indicators, and choosing timeframes.
- Chart type toggles (Bar, Candlestick, Line) and zoom controls.
- Drawing tool activation and quick options access.

![Menu and Toolbar](/img/products/gui/ui-overview/menu_toolbar.png)

## 3. Workspace & Panel Areas

- **Left Panel:** Displays market data, watchlist, and active trade manager panels.
- **Right Workspace:** Houses the primary chart windows and active workspace tabs.

![Workspace Areas](/img/products/gui/ui-overview/workspace_areas.png)

## 4. Status Bar

- Displays active runtime status, network ping/latency, and operational feedback messages.

![Status Bar Areas](/img/products/gui/ui-overview/status_bar_areas.png)

## 5. Strategy & Execution Interfaces

The strategy workspace supports both live execution and historical simulation:

- **Live Trading Dialog:** Used to track and manage real-time active strategy parameters and performance.
  ![Live Trading Dialog](/img/products/gui/ui-overview/livetrade_dialog.png)

- **Backtest Dialog:** Displays historical backtesting details, statistics, and results.
  ![Backtest Dialog](/img/products/gui/ui-overview/backtest_dialog.png)

- **Strategy Input Parameters:** Allows customization of inputs for strategies and custom indicators.
  ![Strategy Input Parameters](/img/products/gui/ui-overview/stategy_inputparams_areas.png)
