# Overview

CLI is the command-line host for running strategies, loading history, and wiring notifiers in automated environments.

## Core Capabilities

- Parse strategy/runtime options from command-line arguments.
- Run in sandbox or production environment modes.
- Configure history storage mode and notifier channels.
- Integrate with core engine and strategy plugins.
