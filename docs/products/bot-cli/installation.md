# Installation

## Requirements

- .NET SDK 10.x
- Linux or Windows host

## Install And Run

- Install packaged bot binary from your release channel.
- Configure required environment variables.
- Start bot with command-line arguments supported by your deployment package.

## Notes

- Ensure strategy DLL and parameter json exist when using --strategy.
- Configure environment variables before startup.
