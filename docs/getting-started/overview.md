# Overview

This project is a trading platform running on .NET 10 that currently supports trading on the OKX exchange (https://www.okx.com), with support for other exchanges coming soon.

It integrates a desktop application (GUI), a CLI application for host-based trading, SDK wrappers, plugin extension points, and strategy engine components. 

## Key Components

- **Desktop App**: A graphical user interface (GUI) to display charts, custom indicators, trade management, TradingView, and market watch.
- **CLI App for Host-Based Trading**: Automated, server-hosted trading with command-line configuration and notification support.
- **SDK/Core Library**: API wrappers, models, storage, a simulation engine, and strategy infrastructure.
- **Plugins**: Loads custom indicator and strategy plugins with version checking capabilities.
