# Tiers

## Community (Free)

Basic features available without a license:
- Chart viewing and basic indicators.
- Market watch and symbol metadata.
- Public data access.

## Pro (Licensed)

Features gated by valid Pro license include:

- Live and paper trading.
- Custom indicator plugin loading.
- Advanced backtest engine.
- Walk-forward and Monte Carlo analysis.
- Telegram/Teams/Webhook notifications.
- Data export and performance reports.

Full feature list maintained in FeatureAccessService.ProFeatures.
