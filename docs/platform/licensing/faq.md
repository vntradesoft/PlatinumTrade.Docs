# Licensing FAQ

## How do I get a license?

Contact the operator with your machine ID from Settings → License. You will receive a .lic file.

## Can I use the same license on multiple machines?

No. Each license is bound to one machine ID.

## What happens when the license expires?

Pro features are locked. Community features remain accessible. A warning is shown before expiration.

## Can I use the app offline after activating?

Yes. Validation is fully offline after import.

## My machine changed hardware — does my license still work?

If the hardware change alters the machine ID computation, you will need a replacement license.
