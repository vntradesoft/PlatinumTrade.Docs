# Platinum Trade — Privacy Policy
This Privacy Policy is incorporated by reference into, and forms part of, the Platinum Trade [Terms of Use](terms-of-use.md). It applies to all use of the Software (as defined in the Terms of Use).

## 1. Local Data Storage
User data, including strategy configurations and trade history, is primarily stored locally on the User's device. Sensitive information such as exchange API credentials may be encrypted at rest using OS-level protections (e.g., Windows DPAPI).

## 2. No Unauthorized Transmission
The Software does not intentionally transmit trading activity to Platinum Trade's servers unless explicitly disclosed and consented to (for example, optional license verification or crash reporting).

## 3. Data Protection Regulations (e.g., GDPR)
If the User resides in a jurisdiction with specific data protection requirements (e.g., the EU/EEA under GDPR), the User acknowledges that minimal technical data may be processed strictly as necessary to operate the Software, and may contact us at the email below for any data-related inquiry.

## 4. Security Responsibility
While the Software applies OS-level protections to sensitive local data, the User remains responsible for safeguarding their own device, API keys, passwords, license files, and backups. See the [Limitation of Liability](liability.md) for further detail.
