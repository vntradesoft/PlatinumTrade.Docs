# Platinum Trade Documentation

Welcome to the documentation portal for **Platinum Trade**, a comprehensive algorithmic trading ecosystem designed for cryptocurrency traders, system integrators, and strategy developers.

The Platinum Trade ecosystem consists of three main components:

---

## 1. Platinum Trade GUI (Desktop App)
A professional desktop application designed for manual trading, charting, visual strategy configuration, and running local backtests.
- **Get Started**: [GUI Installation Guide](./products/gui/installation.md)
- **UI Overview**: [Understanding the Workspace](./products/gui/ui-overview.md)
- **Trading Basics**: [Live and Paper Trading Flows](./products/gui/trading-basics.md)
- **Settings**: [API Credentials & Configurations](./products/gui/settings/api-credentials.md)

## 2. Platinum Trade Bot (CLI)
A lightweight command-line host designed for running automated trading strategies in headless or server environments (Linux/Windows).
- **Overview**: [Bot CLI Usage](./products/bot-cli/overview.md)
- **Linux Deployment**: [Running as a Systemd Service](./products/bot-cli/run-on-ubuntu.md)

## 3. Developer SDK & API
An open-source SDK (`Pt.Okx.Sdk`) allowing developers to extend Platinum Trade by building custom technical indicators and automated trading strategies.
- **SDK Overview**: [Integrating the SDK](./sdk/intro.md)
- **Indicator API**: [Developing Custom Indicators](./sdk/indicator/overview.md)
- **Strategy API**: [Writing Automated Strategies](./sdk/strategy/overview.md)
- **Drawing API**: [Rendering Visual Shapes on Charts](./sdk/drawing-api.md)
