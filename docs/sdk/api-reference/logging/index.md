# Logging API

The Logging API allows your strategies to output messages to the user interface, typically the Experts or Journal tab, useful for debugging and audit trails.

| Interface / Class | Description |
|---|---|
| [`ILoggerClient`](./logger-client.md) | Interface for logging messages. |
| [`LogLevel`](../../enums.md#loglevel) | Enumeration defining the severity of the log. |
