# Storage API

The Storage API allows your strategies to persist state across restarts or save configuration to the local disk.

| Interface / Class | Description |
|---|---|
| [`IStorageClient`](./storage-client.md) | Interface for storing and retrieving key-value pairs or files. |
| [`StorageResult`](../../models.md#storageresult) | Result of a storage operation. |
