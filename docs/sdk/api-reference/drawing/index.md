# Drawing API

The Drawing API allows your strategies to draw visual objects on the chart, such as trend lines, text, fibonacci retracements, and rectangles.

| Interface / Class | Description |
|---|---|
| [`IDrawingClient`](./drawing-client.md) | Interface for managing drawing objects on the chart. |
| [`DrawingObject`](../../models.md#drawingobject) | Represents a base drawing object on the chart. |
| [`IDrawable`](./drawable.md) | Interface for anything that can be drawn. |
