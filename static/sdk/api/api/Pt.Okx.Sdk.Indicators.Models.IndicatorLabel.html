Color

Gets or sets the fallback color for this plot. Used when ColorIndexBuffer is not set or yields no valid palette entry. Default is Blue.

public IndicatorColor Color { get; set; }

Property Value

IndicatorColor

ColorIndexBuffer

Gets or sets the optional buffer index that contains per-point color indexes for this plot. The referenced buffer must be registered as ColorIndex. When null, Color is used uniformly for every point. Each indicator label may reference a different color-index buffer, so a multi-line indicator can have each line change color independently.

public int? ColorIndexBuffer { get; set; }

Property Value

int?

ColorPalette

Gets the color palette used when ColorIndexBuffer is set. The key is an integer color index written into the color buffer by OnCalculate; the value is the IndicatorColor to render for that index. Points whose index is not present in the palette fall back to Color.

public IDictionary<int, IndicatorColor> ColorPalette { get; set; }

Property Value

IDictionary<int, IndicatorColor>

Examples

ColorPalette = new Dictionary<int, IndicatorColor>
{
    { 0, IndicatorColor.Red },   // bearish
    { 1, IndicatorColor.Lime }   // bullish
}
