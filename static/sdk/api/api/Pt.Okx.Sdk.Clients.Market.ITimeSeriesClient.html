    CopySeries(string?, Timeframe?, int, int)

    Copies an array of OHLCV data by start position and number of candles.

    Declaration
    Task<CandleData[]> CopySeries(string? symbol = null, Timeframe? timeframe = null, int startPos = 0, int count = 1)
    Parameters
    Type Name Description
    string symbol

    The trading symbol (optional).
    Timeframe? timeframe

    The timeframe for the data (optional).
    int startPos

    The starting position (index) in the series.
    int count

    The number of candles to copy.
    Returns
    Type Description
    Task<CandleData[]>

    An array of OHLCV candle data starting from the specified position.
    Remarks

    Time boundaries are normalized to the requested timeframe. In backtest mode, if the current open candle timestamp falls inside the requested window, that open candle can be appended as the last item.

    Strict behavior:

    • Empty result when start or end boundary is not available in loaded data.
    • Returns data only when both normalized boundaries exist in the returned window.

    The same strict boundary rule is inherited by component-copy APIs that rely on this method (for example CopyTimes(string?, Timeframe?, int, int), CopyOpens(string?, Timeframe?, int, int), CopyHighs(string?, Timeframe?, int, int), CopyLows(string?, Timeframe?, int, int), CopyCloses(string?, Timeframe?, int, int), and CopyVolumes(string?, Timeframe?, int, int)).

    UpdateOpenCandleIndicators(string?, Timeframe?)

    On-demand: recalculate indicators using the current open (forming) candle data. Follows a lazy evaluation pattern — indicators are only recalculated when explicitly requested, not automatically on every tick.

    After calling this method, indicator buffers' last element reflects the open candle values. When the bar closes, the normal close flow overwrites it with the closed candle automatically.
