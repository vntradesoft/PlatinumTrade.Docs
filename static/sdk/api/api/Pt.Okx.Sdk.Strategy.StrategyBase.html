    Syntax
    public abstract class StrategyBase : IStrategy
    Remarks

    Inherit from StrategyBase and override OnTickAsync(TickPhase, CancellationToken) (required) plus any optional event handlers needed by your strategy.

    Event dispatch and state mutation are handled by the hosting engine.
