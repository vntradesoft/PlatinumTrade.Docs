GetTrend(int, double)

Determines the moving average trend direction over a specified lookback period.

MATrendDirection GetTrend(int lookback = 5, double tolerance = 1E-06)

Parameters

lookback int

The number of most recent data points to consider when calculating the trend. Default is 5.

tolerance double

The minimum absolute difference required to consider the trend as upward or downward. Default is 1e-6.

Returns

MATrendDirection

A value indicating whether the trend is upward, downward, or flat.
