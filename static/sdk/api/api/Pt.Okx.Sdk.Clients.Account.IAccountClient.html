    Interface IAccountClient

    Provides access to account information, balances, trading configuration, and account performance metrics.

    Namespace: Pt.Okx.Sdk.Clients.Account
    Assembly: Pt.Okx.Sdk.dll
    Syntax
    public interface IAccountClient
    Remarks

    Currently supports OKX USDT-margined perpetual swap accounts only. Spot, Futures, Options, and other instrument types are not supported.
