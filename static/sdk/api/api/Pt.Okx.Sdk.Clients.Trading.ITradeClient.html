    Interface ITradeClient

    Provides trading operations including order placement, order management, position management, and trade history retrieval.

    Namespace: Pt.Okx.Sdk.Clients.Trading
    Assembly: Pt.Okx.Sdk.dll
    Syntax
    public interface ITradeClient
    Remarks

    Currently supports OKX USDT-margined perpetual swap instruments only. Spot, Futures, Options, and other instrument types are not supported.
