    LoadFromDefaults(IReadOnlyDictionary<string, InputParameter>, string)

    Loads parameters from code-defined defaults, merging with an existing JSON file if present. If the JSON file exists, user-modified values are preserved for matching keys. If the JSON file does not exist, or the schema has changed (keys differ), a new JSON file is written with merged data.
