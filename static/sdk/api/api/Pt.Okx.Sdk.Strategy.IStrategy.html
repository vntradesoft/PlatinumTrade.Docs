    Interface IStrategy

    Defines the lifecycle and execution contract for a trading strategy.

    Namespace: Pt.Okx.Sdk.Strategy
    Assembly: Pt.Okx.Sdk.dll
    Syntax
    public interface IStrategy
    Remarks

    A strategy is initialized once, then executed repeatedly by the hosting engine, and eventually stopped when no longer needed.

    For most use cases, inherit from StrategyBase instead of implementing this interface directly.

    The engine owns event dispatch and state mutation. Strategies receive market cadence via TickPhase and may override optional event handlers in StrategyBase.
